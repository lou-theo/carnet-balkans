# Carnet Balkans

Carnet de voyage web (page unique) — **Europe centrale → Balkans**, ~30 juillet → 14 août 2026, sans voiture.

Site live : https://lou-theo.github.io/carnet-balkans/

Site statique : un seul fichier `index.html` (HTML + CSS + JS inline, aucune dépendance). Publié via GitHub Pages.

## Plan v2 (juillet 2026)

Le voyage a pivoté. Ancien plan (Serbie · Bosnie · Monténégro, solo, 16 j) → nouveau plan :

- **Solo avant Budapest** : Pologne (Cracovie) + Tatras (polonaises et/ou slovaques) — remplace le Monténégro perdu.
- **Retrouvailles à Budapest ~4 août** avec une amie arrivant de Hongrie.
- **Ensemble** : Budapest → Belgrade, puis **Bosnie selon le temps restant**.
- **Monténégro abandonné.**

Zones à distinguer : **Pologne · Slovaquie · Hongrie = UE + Schengen** (CNI seule, roaming FR, CEAM valable) ; **Serbie · Bosnie = hors UE** (passeport conseillé, eSIM, assurance dédiée).

**Encore à figer** : durées de chaque étape, aéroport de retour (Belgrade ? Sarajevo ?), Bosnie oui/non. La carte et l'itinéraire jour-par-jour du site sont marqués « en refonte » tant que ce n'est pas calé.
