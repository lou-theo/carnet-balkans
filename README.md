# Carnet Balkans

Carnet de voyage web (page unique) — Serbie · Bosnie · Monténégro, 16 jours, sans voiture.

Site live : https://lou-theo.github.io/carnet-balkans/

Site statique : un seul fichier `index.html` (HTML + CSS + JS inline, aucune dépendance). Publié via GitHub Pages.

## Modifier

Éditer `index.html`, commit, push. GitHub Pages redéploie automatiquement.

## Budget (16 jours, profil confortable — estimations)

Auberges sociales + quelques privés et excursions.

| Poste | Estimation |
|---|---|
| Vols open-jaw (Belgrade + Tivat) | 350–550 € |
| Transferts France (Reims↔Paris + aéroport) | 50–90 € |
| Auberges (15 nuits, ~18–25 €) | 270–450 € |
| Repas & boissons (~20–32 €/j) | 320–510 € |
| Bus + trains intercités | 90–130 € |
| Activités & excursions | 200–300 € |
| **Total** | **~1 300–2 000 €** |
